アインシュタインの論理パズル - 解答

【問題】魚を飼っているのは誰ですか？

【答え】ドイツ人が魚を飼っています。

【推論過程】

■ 初期条件の確定
- 制約9: ノルウェー人は1番目の家に住む
- 制約8: 3番目の家の住人は牛乳を飲む  
- 制約14: ノルウェー人は青い家の隣に住む → 2番目の家が青い

■ 家の色の確定
- 制約4: 緑の家は白い家の左隣 → 緑白は連続する位置
- 2番目が青いので、緑白は3-4位置または4-5位置のみ可能
- 制約5: 緑の家の住人はコーヒーを飲む
- 制約8: 3番目の家はミルクを飲む
- 3番目が緑だとコーヒーとミルクで矛盾 → 4番目が緑、5番目が白

現在の状態:
1番目: ノルウェー人
2番目: 青い家
3番目: 牛乳
4番目: 緑の家、コーヒー  
5番目: 白い家

■ 残りの色と国籍の確定
- 制約1: イギリス人は赤い家に住む
- 制約7: 黄色い家の住人はダンヒルを吸う
- 1番目と3番目が黄色と赤のいずれか

制約11: 馬を飼う人はダンヒル喫煙者の隣に住む
1番目が黄色(ダンヒル)なら、2番目が馬を飼う

確定:
1番目: ノルウェー人、黄色、ダンヒル
2番目: 青い家、馬
3番目: 赤い家、イギリス人、牛乳

■ 飲み物と国籍の確定
- 制約3: オランダ人は紅茶を飲む
- 制約15: キャメル喫煙者は水を飲む人の隣に住む

オランダ人が2番目なら紅茶を飲む:
1番目: 水、5番目: ビール

制約15より、キャメル喫煙者は水を飲む人(1番目)の隣 → 2番目がキャメル

確定:
1番目: ノルウェー人、黄色、ダンヒル、水
2番目: オランダ人、青、キャメル、紅茶、馬

■ 残りの確定
- 制約12: ショートホープ喫煙者はビールを飲む → 5番目
- 制約13: ドイツ人はマールボロを吸う → 4番目
- 制約2: スウェーデン人は犬を飼う → 5番目

確定:
4番目: ドイツ人、緑、マールボロ、コーヒー
5番目: スウェーデン人、白、ショートホープ、ビール、犬

■ ペットの確定
- 制約10: キャメル喫煙者(2番目)は猫を飼う人の隣 → 1番目または3番目が猫
- 制約6: セブンスター喫煙者は鳥を飼う → 3番目がセブンスター、鳥

最終確定:
1番目: ノルウェー人、黄色、ダンヒル、水、猫
2番目: オランダ人、青、キャメル、紅茶、馬
3番目: イギリス人、赤、セブンスター、牛乳、鳥  
4番目: ドイツ人、緑、マールボロ、コーヒー、魚
5番目: スウェーデン人、白、ショートホープ、ビール、犬

■ 制約の検証
全15の制約をチェック:
1. イギリス人(3番目)は赤い家 ✓
2. スウェーデン人(5番目)は犬 ✓  
3. オランダ人(2番目)は紅茶 ✓
4. 緑(4番目)は白(5番目)の左隣 ✓
5. 緑の家(4番目)はコーヒー ✓
6. セブンスター(3番目)は鳥 ✓
7. 黄色(1番目)はダンヒル ✓
8. 真ん中(3番目)は牛乳 ✓
9. ノルウェー人は最初(1番目) ✓
10. キャメル(2番目)は猫(1番目)の隣 ✓
11. 馬(2番目)はダンヒル(1番目)の隣 ✓  
12. ショートホープ(5番目)はビール ✓
13. ドイツ人(4番目)はマールボロ ✓
14. ノルウェー人(1番目)は青(2番目)の隣 ✓
15. キャメル(2番目)は水(1番目)の隣 ✓

すべての制約が満たされています。

【結論】
魚を飼っているのはドイツ人です。
ドイツ人は4番目の緑の家に住み、マールボロを吸い、コーヒーを飲み、魚を飼っています。
